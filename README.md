# AI 小说转剧本工具

本项目是一个使用 Python 开发的 AI 辅助剧本创作工具，目标是帮助小说作者将 3 个章节以上的小说文本转换为结构化 YAML 剧本初稿。

## 项目目标

很多小说作者希望将自己的作品改编成剧本，但从小说到剧本需要重新整理人物、场景、情节和对白。本工具希望降低改编门槛，让作者可以快速获得一个可编辑、可继续打磨的剧本初稿。

## 当前版本功能

- 使用 Streamlit 搭建基础网页界面
- 支持输入小说文本
- 提供“生成剧本”按钮
- 显示 YAML 剧本示例输出

## 后续计划

- 增加小说章节检测功能
- 设计完整的剧本 YAML Schema
- 接入 AI 生成剧本内容
- 支持 YAML 编辑和下载
- 编写 Schema 设计说明文档
